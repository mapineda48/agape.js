# CRM y Gestión Comercial

El módulo Comercial de Agape.js permite centralizar la relación con sus clientes y gestionar el ciclo de vida de las ventas, desde la creación del pedido hasta su seguimiento operacional.

## Secciones del Módulo

### [Administración de Clientes](./clientes.md)
Centralice la información de sus clientes, incluyendo datos legales, comerciales y de contacto.

### [Pedidos de Venta](./pedidos.md)
Gestió los compromisos de venta con sus clientes, controle estados de despacho y facturación.

---

## Integración con otros Módulos
*   **Inventarios:** Los pedidos de venta reservan mercancía y consultan la disponibilidad en tiempo real.
*   **Finanzas:** La configuración comercial del cliente (listas de precios, días de crédito) automatiza la creación de facturas y control de cartera.
*   **Configuración:** Utiliza las series de numeración para el control de documentos comerciales.
