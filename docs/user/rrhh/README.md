# Recursos Humanos (RRHH)

El módulo de Recursos Humanos de Agape.js permite gestionar el talento humano de su organización, organizando la estructura jerárquica y manteniendo un registro detallado de cada colaborador.

## Secciones del Módulo

### [Gestión de Empleados](./empleados.md)
Mantenga la ficha técnica de sus colaboradores, incluyendo datos personales, fechas de contratación y perfiles de acceso.

### [Estructura Organizacional (Departamentos)](./departamentos.md)
Defina los departamentos y áreas de la empresa, estableciendo jerarquías y centros de costos.

### [Cargos y Posiciones](./cargos.md)
Administre el catálogo de cargos disponibles en la empresa, definiendo niveles y descripciones de puesto.

---

## Integración con otros Módulos
*   **Inventarios:** El personal de almacén registrado como empleado es el responsable de autorizar los movimientos de entrada y salida de mercancía.
*   **Finanzas:** Los departamentos permiten asociar gastos y nómina a centros de costos específicos para un control presupuestal detallado.
*   **Seguridad:** Cada empleado está vinculado a un usuario del sistema, permitiendo la trazabilidad de quién realiza cada acción en la plataforma.
