# Cargos y Posiciones

El catálogo de cargos define los roles específicos que pueden desempeñar los colaboradores en la empresa.

## Configuración del Cargo

Cada posición registrada en el sistema cuenta con:
*   **Código:** Abreviatura única (ej: ANALIST-CONT).
*   **Nombre:** Título oficial del cargo (ej: Analista Contable Senior).
*   **Nivel:** Rango jerárquico o grado de la posición, útil para escalas salariales y niveles de autoridad.
*   **Descripción:** Detalle de las responsabilidades o perfil requerido para el cargo.

## Estados de los Cargos
*   **Habilitado:** El cargo está disponible para ser asignado a nuevos empleados o vacantes.
*   **Deshabilitado:** El cargo ya no se utiliza en la organización. El sistema permite mantener el historial pero impide nuevas asignaciones.

## Importancia de la Estandarización
Mantener un catálogo de cargos limpio permite:
1.  **Análisis de Cargos:** Comparar funciones y perfiles de forma uniforme entre diferentes departamentos.
2.  **Escalabilidad:** Facilitar el proceso de contratación al tener perfiles ya predefinidos en el sistema.
3.  **Auditoría:** Conocer exactamente qué roles existen en la empresa independientemente de las personas que los ocupan.
