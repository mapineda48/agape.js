# Mi Perfil y Seguridad

Esta sección permite la administración de los usuarios del sistema y la configuración de sus niveles de acceso.

## Gestión de Usuarios
Cada colaborador que interactúe con el sistema debe tener una cuenta individual.

*   **Identificación:** Nombre completo y correo electrónico.
*   **Estado del Usuario:** Determina si el usuario puede o no ingresar al sistema.

## Roles y Permisos
El sistema utiliza un esquema de permisos para controlar qué acciones puede realizar cada usuario:
*   **Lectura:** Permite ver información.
*   **Escritura:** Permite crear y modificar registros.
*   **Anulación/Eliminación:** Permite cancelar operaciones (restringido usualmente a supervisores).

## Seguridad de la Cuenta
*   **Contraseñas:** Se recomienda el uso de contraseñas fuertes y no compartirlas.
*   **Cierre de Sesión:** Siempre cierre su sesión al terminar de usar el sistema en computadores compartidos.

> *Nota: Para cambios avanzados en permisos, contacte al administrador del sistema.*
