# Configuración General

En este módulo se gestionan los parámetros base que permiten el correcto funcionamiento de los demás módulos del sistema. Estos ajustes suelen configurarse al inicio de la implementación o cuando hay cambios estructurales en la operación.

## Secciones

### [Unidades de Medida](./unidades-medida.md)
Define las magnitudes en las que se expresan las cantidades de los productos (ej: Kilogramo, Unidad, Metro). Es vital para el control de inventarios y conversiones.

### [Series de Numeración](./numeracion.md)
Administra los consecutivos y prefijos de los documentos legales y administrativos del sistema, como Facturas, Pedidos u Órdenes de Compra.

### [Mi Perfil y Seguridad](./seguridad.md)
Gestión de usuarios internos, perfiles de acceso y preferencias personales dentro de la plataforma.
