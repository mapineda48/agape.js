# Web Development Guidelines

When creating or modifying frontend components, ensure compliance with the following documentation:

- **Contributing:** Follow the project standards in `docs/develop/CONTRIBUTING.md`.
- **Forms:** Follow the standards defined in `docs/develop/web/form.md` for form management.
- **Routing:** Refer to `docs/develop/web/router.md` when defining or updating application routes.
- **Testing:** Consult `docs/develop/web/test.md` for best practices regarding React SPA testing.
- **Modals & Notifications:** Adhere to `docs/develop/web/modal.md` when implementing modals or notifications.
- **DTOs:** Follow `docs/develop/dto.md` for Data Transfer Objects shared between the backend (`app`) and frontend (`web`).
